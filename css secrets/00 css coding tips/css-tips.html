尽量减少改动时要编辑的地方

1.当某些值相互依赖时，就应该把他们的依赖关系用代码表现出来。

.button {
	font-size: 20px;
	line-height: 30px;
}
-->> 
.button {
	font-size: 20px;
	line-height: 1.5;
}

为什么还要把字号定位绝对长度那？

.button {
	font-size: 20px;
	line-height: 1.5;
}
-->> 
.button {
	font-size: 125%; /* 假设父级字号是16px;*/
	line-height: 1.5;
}

改变字号时，按钮尺寸随之变化，但是并不协调，不有随之缩放。改成em单位，使得依赖字号随之缩放。

注意：审视下，border 的宽度不应该随之变化。

.button {
	padding: 6px 16px;
	border: 1px solid #446d88;
	border-radius: 4px;
	box-shadow: 0 1px 5px grey;
	text-shadow: 0 -1px 1px #335166;
	font-size: 20px;
	line-height: 30px;
}
-->> 
.button {
	padding: 0.3empx 0.8em;
	border: 1px solid #446d88;
	border-radius: 0.2em;
	box-shadow: 0 05em .25em grey;
	text-shadow: 0 -.05em .05em #335166;
	font-size: 20px;
	line-height: 1.5;	
}
