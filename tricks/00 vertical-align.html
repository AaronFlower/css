方法1 Line-height , Single Line

单行文本，图片利用line-height来完成垂直居中。

html

	
		<div class="parent">
			<div class="child">Vertical Align Text</div>
		</div>

css

	
		.child {
			line-height: 200px;
			text-align: center;
		}

Centering img with Line-height

对于图片元素也可以就应用该方法，确保父元素的line-height比图片的高度要高。

html

	
		<div id="img-parent">
			<img src="230_152A.jpg" alt="" />
		</div>

css

	
		#img-parent {
			line-height: 300px;
			text-align: center;
		}
		#img-parent {
			vertical-ailgn: middle;
		}

方法2： CSS Table Method

通过设置元素的及子元素的display属性来实现。

html

	
		<div id="parent-2">
			<div id="child-2">Align-Content Here </div>
		</div>

css

	
		#parent-2 {
			display: table;
			height: 130px;
			width: 140px;
		}
		#child-2 {
			display: table-cell;
			vertical-align: middle;
			text-align: center;
		}

方法3: 绝对定位 + 负外边距 (Absolute Positioning and Negative Margin)

html

	
		<div id="parent-3">
			<div id="child-3"></div>
		</div>

css

	
		#parent-3{
			position: relative;
			width: 200px;
			height:200px;
		}
		#child-3 {
			position: absolute;
			top: 50%;
			left: 50%;
			width:50%;
			height: 30%;
			margin: -15% 0 0 -25%;
		}

方法4: 绝对定位 + 拉伸 (Absolute Positioning and Stretching)

html

	
		<div id="parent-4">
			<div id="child-4">
				Align-Content
			</div>
		</div>

CSS

	
		#parent-4 {
			position: relative;
			height: 200px;
			width: 200px;
			background-color: #ccc;
		}

		#child-4 {
			position: absolute;
			top: 0;
			bottom: 0;
			left: 0;
			right: 0;
			margin:auto;
