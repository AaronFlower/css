单个 Block 元素

对于Block 元素，可以设置 margin-left/right 为 auto; 即可， 并且设置一个宽度，如果不设置宽度的话，会平铺所有的空间。

多个 Block 元素

对于多个 Block 元素们当然可以直接让这些 block 元素 inline-block 化，然后让元素 display: center; 就可以了。 但是我们还可以使用父元素 display: flex; justify-content: center;来完成。

Vertically Centering

inline or inline-elements

Is it single line?

1. 对于单行 inline 元素，可以使 padding-top/padding-bottom 相同来完成。

We are Centered Bits of Text

2. 如果无法使用 padding, 我们可以使用 line-height 与 height 相等来完成。

I'm a vertical centered line by the same value of line-height and height.

Is it multiple lines?

同样，也可使用 padding 来完成。但是如果 padding 不适用的话，我们可以使用下面几种方法：

1. 使用 table-cell 的 vertial-align 来解决。

I'm vertically centered multiple lines of text in a CSS-created table layout.

2. 使用 flexbox 来完成。

I'm vertically centered multiple lines of text by flexbox layout.

3. 如果 padding, table-cell, flexbox 都不适用，那么可以使用 ghost pseudo-element 来完成。
