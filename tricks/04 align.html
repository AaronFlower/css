单个 Block 元素

对于Block 元素，可以设置 margin-left/right 为 auto; 即可， 并且设置一个宽度，如果不设置宽度的话，会平铺所有的空间。

I'm a block Element;
I'm a block Element has fixed width;

多个 Block 元素

对于多个 Block 元素们当然可以直接让这些 block 元素 inline-block 化，然后让元素 display: center; 就可以了。 但是我们还可以使用父元素 display: flex; justify-content: center;来完成。
